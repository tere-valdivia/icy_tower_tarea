IMPORTANTE: este programa esta implementado con Python 2.7. Usar Python 2.x para su ejecucion

************
Para correr escena inicial (parte 1 de la tarea): Correr archivo Escena_inicial.py usando su IDE o desde consola:
>>> python Escena_inicial.py

Para correr Demo (partes 2 y 3 de la tarea): Correr archivo Escena_mecanica.py usando su IDE o desde consola:
>>> python Escena_mecanica.py

La demo dura 30 segundos y las 50 plataformas son alcanzables. Si se acaba el juego, puede presionar la tecla 'p' para volver al inicio.
